# UMAP — YouTube Map Explorer

**Live Demo:** https://umapo.netlify.app/

Discover location-based YouTube videos on an interactive map. Search any city, pan and zoom to explore, and filter results by topic, recency, and sort order.

## Created By

**Sharjeel Khan & Luke Kustra**

## Tech Stack

- **Frontend:** React, Vite
- **Backend:** Node.js, Express
- **Map:** Leaflet, OpenStreetMap
- **APIs:** YouTube Data API v3, Nominatim Geocoding

---

## Features

- Search any city or place
- Explore videos directly on an interactive map
- Pan and zoom to load videos for the visible area
- Clustered markers for dense areas
- Sidebar video list with matching map pins
- Filter by keyword, recency, and sort order
- Video detail panel with direct YouTube link
- Optional geolocation on first load
